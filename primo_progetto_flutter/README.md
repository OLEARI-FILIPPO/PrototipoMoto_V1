# PrototipoMoto_V1 — App Flutter

App Flutter per la navigazione in gruppo di motociclette con comunicazione mesh ESP32-S3 via BLE.

## Documentazione

- **Logica Leader/Follower e navigazione in gruppo** → [`../docs/NAVIGATION_LOGIC.md`](../docs/NAVIGATION_LOGIC.md)
- **Guida utente completa** → [`GUIDA_UTENTE.md`](GUIDA_UTENTE.md)

## Getting Started

### Prerequisiti

- Flutter SDK 3.9.2 o superiore
- Android 5.0+ (API 21) o iOS 12.0+
- Un dispositivo ESP32-S3 con il firmware di `sketch_ESP32_S3_BLE_Server/`

### Installazione

```bash
flutter pub get
flutter run
```

Per istruzioni dettagliate su funzionalità, permessi e compilazione consulta [`GUIDA_UTENTE.md`](GUIDA_UTENTE.md).
